## 🚀 About the Project

This repository is a **Spring Boot learning journey** for mastering backend development using Java.

It includes practical examples of building REST APIs, working with databases, and understanding core Spring Boot concepts.

---

## 🌟 Features

- 🌱 Spring Boot project setup  
- 🔗 REST API development  
- 🧩 Dependency Injection (IoC)  
- 📦 Maven build management  
- 🗄️ CRUD operations with database  
- ⚠️ Exception handling & validation  
- 🧪 API testing with Postman  

---

## 🛠️ Tech Stack

- ☕ Java 17  
- 🌱 Spring Boot  
- 📦 Maven  
- 🗄️ MySQL / H2 Database  
- 📬 Postman
